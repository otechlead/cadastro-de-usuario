# 🐞 Projeto de Cadastro de Usuário (com Bugs Intencionais)

Este repositório contém uma aplicação simples de cadastro de usuário desenvolvida com HTML, CSS e JavaScript. O objetivo principal do projeto não é ser um sistema perfeito, mas sim servir como um ambiente de prática para identificação e correção de bugs.

## 🎯 Objetivo

O projeto foi criado propositalmente com possíveis falhas para que desenvolvedores possam:

- Identificar bugs no código
- Praticar debugging
- Aplicar boas práticas de desenvolvimento
- Refatorar código
- Melhorar validações e segurança

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## ⚠️ Aviso Importante

Este projeto contém bugs intencionais, como por exemplo:

- Validações incompletas ou incorretas
- Possíveis falhas de lógica
- Problemas de UX/UI
- Código não otimizado
- Más práticas propositalmente inseridas

## 🧪 Desafio

Tente encontrar e corrigir:

- ❌ Erros de validação de formulário
- ❌ Problemas de manipulação do DOM
- ❌ Falhas de segurança (ex: inputs não tratados)
- ❌ Código redundante ou mal estruturado
- ❌ Bugs de usabilidade

## 🚀 Como usar

Siga os passos abaixo para executar o projeto e contribuir:

1. Clone o repositório:
```bash
git clone https://github.com/otechlead/cadastro-de-usuario.git
cd cadastro-de-usuario
```
2. Abra o arquivo index.html em seu navegador
3. Explore a aplicação e tente identificar possíveis bugs
4. Caso encontre problemas:
- Abra uma issue descrevendo o bug
- Proponha uma solução
5. Para contribuir com correções:
- Faça um fork do projeto
- Crie uma nova branch (git checkout -b minha-correcao)
- Commit suas alterações (git commit -m "Corrige bug X")
- Envie para o repositório (git push origin minha-correcao)
- Abra um Pull Request
